Location

This is an interesting town with some good restaurants, as well as several archaeological sites.

Docking

It is a large commercial port, but, to the east of this, there is a well protected marina.

Visitors berth in the southern part of this inner harbour.

As a nearby alternative, Stintino offers several sheltered marinas and is a useful base for exploring I.Asinara.
